# loan-emi-excel-project
# 📊 Banking Loan EMI Calculator & Amortization Dashboard

A complete **Excel-based Banking Loan Analytics Project** that calculates EMI, generates amortization schedules, performs loan portfolio analysis, and visualizes banking data using interactive dashboards, Pivot Tables, and Pivot Charts.

---

# 🚀 Project Overview

This project simulates a mini banking analytics system using **Microsoft Excel**.
It helps analyze customer loan data, calculate EMIs automatically, track repayment schedules, and generate interactive financial dashboards.

The project demonstrates:

* Financial Modeling
* EMI Calculation
* Loan Amortization
* Data Cleaning
* Dashboard Design
* Pivot Table Analysis
* Advanced Excel Functions
